Dust is Java-based web framework built around components, templates and JAX-RS.

Namespaced Components, pages and assets all live within namespaces allowing them to be reused.
Component driven Common layout and functionality can be packaged as components that can be used from templates.
Developer friendly A special development mode makes seeing changes as simple as refreshing your browser.
Awesome templates Templates are created via XML and get procssed into HTML at runtime.
Based on JAX-RS Dust uses JAX-RS making it easy to define the web structure you need.
Integration Namespaces, components and templates all help with making Dust easy to integrate with your existing solution.
